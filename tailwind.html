CSS ramverk - Tailwindcss




Vad är Tailwindcss?


Tailwind CSS släpptes först i maj 2019 och är ett front-end CSS-ramverk. Tailwind är ett av de mest populära CSS-ramverken på marknaden. Dess funktioner gör den till ett idealiskt val för ett brett utbud av projekt. Den största skillnaden mellan Tailwind och dess konkurrenter är att det ger utvecklare full kontroll över stilarna för webbapplikationer. Du kan läsa mer om Tailwindcss här.

Fördelar med Tailwindcss


Det finns inget snabbare ramverk än Tailwind när det kommer till HTML-styling. Som ett resultat kan man enkelt skapa snygga layouter genom att direkt styla element. Detta är möjligt eftersom Tailwind erbjuder tusentals inbyggda klasser som inte kräver att man skapar projekt från grunden. Med Tailwinds färdiga klasser kan man designa layouten direkt i HTML-filen. Detta gör det till ett mycket lyhört, mobilvänligt CSS-ramverk.

Nackdelar med Tailwindcss


På grund av de inbyggda CSS-klasserna är Tailwind ganska intensivt att lära sig. Till skillnad från Bulma och Bootstrap har Tailwind inte många betydande stylingelement. Tyvärr betyder detta att man manuellt måste lägga till funktioner som rubriker, knappar och navigeringsfält för webbappar. Dokumentationen i Tailwindcss är inte lika bra och omfattande som den som finns till till exempel Bootstrap.

Mina tankar om Tailwindcss


Jag upplever Tailwindcss som mycket krångligare än Bootstrap och tycker att den är säkert mycket trevlig för personer som gillar att jobba med css men som vill göra saker lite snabbare. Jag har alltid problem med design och saknar fantasi för att komma på hur saker och ting ska se ut för att det blir snyggt - därför uppskatar jag Bootstraps färdiga lösningar mer. Även om det är bra att känna till Tailwindcss och ha ett hum om hur den ska användas så blir det nog inte min favorit.
